# Dynamic Quote Generator

## Description
This project demonstrates **advanced DOM manipulation** in JavaScript. Users can:
- View random quotes
- Add new quotes dynamically with a category
- See quotes update immediately on the page

## Files
- `index.html` — Main HTML structure
- `script.js` — JavaScript file handling dynamic content
- `README.md` — Project documentation

## How to Use
1. Open `index.html` in a browser.
2. Click **Show New Quote** to view a random quote.
3. Add your own quotes using the input fields and click **Add Quote**.
